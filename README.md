# Tabela de Preços Interativa

## Descrição
<p>
Este sistema foi desenvolvido com o Python e o Django Framework, ele utiliza o Django Admin para gerenciar a 
listagem de produtos por categoria na pagina index do projeto. 
</p>

## Requisitos

 - Python 3.10.9
 - Django 4.1.7
